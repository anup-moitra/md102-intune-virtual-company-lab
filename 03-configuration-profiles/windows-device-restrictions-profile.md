# Windows Device Restrictions Profile

## Status

Planned / Optional

## Objective

Create a Windows device restrictions profile to test controlled device settings in Microsoft Intune.

## Planned Scope

This lab will be completed after core enrollment, compliance, Conditional Access, application deployment, and endpoint security basics are completed.

Possible settings to test later:

- Control Panel restrictions
- Personalization restrictions
- Local device restrictions
- Microsoft Edge restrictions
- Removable storage restrictions

## Safety Note

This lab should be tested with a pilot group first. Do not apply strict restrictions broadly without testing.
