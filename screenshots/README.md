# Screenshots

This folder stores sanitized screenshots for the MD-102 Intune virtual company lab.

## Important

Do not upload screenshots that show:

- Full real email addresses
- Tenant IDs
- Device IDs
- Serial numbers
- Autopilot hardware hashes
- BitLocker recovery keys
- Passwords
- MFA QR codes
- Request IDs
- Correlation IDs
- Internal IP addresses
- Production data

Use the subfolders under `screenshots/sanitized/` for public-safe evidence.
